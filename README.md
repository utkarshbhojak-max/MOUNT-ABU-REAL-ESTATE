# 🏔️ Mount Abu Real Estate Website

A modern, responsive, and high-performance landing page for **Mount Abu Real Estate**, built to showcase local property expertise in Mount Abu, Rajasthan. This project was developed as a professional tool to connect buyers and sellers with a focus on trust and 30+ years of local experience.

## ✨ Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views using CSS Media Queries.
- **Dynamic Animations:** Features smooth fade-in effects, floating card animations, and a linear-gradient text "pop" effect for a premium feel.
- **One-Click Communication:** - Integrated **WhatsApp API** that pre-fills a message with the user's name and contact info.
  - Direct `tel:` links for instant calling.
- **Interactive Mapping:** Embedded Google Maps pinpointing the office location at **MDR60, Mount Abu** with a "Single-Click" navigation button.
- **Modern UI/UX:** Built with the **Inter** font family and a clean, "Teal & Slate" color palette.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Vanilla)
- **Icons & Graphics:** Inline SVG for high-speed loading and crisp resolution.
- **Fonts:** Google Fonts (Inter)
- **Deployment:** GitHub Pages
